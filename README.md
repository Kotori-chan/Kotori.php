## Intro

Kotori.php -- a Tiny Model-View-Controller (MVC) PHP Framework

## Installation

~~~bash
$ composer require kokororin/kotori-php
# via git
$ git clone https://github.com/kokororin/Kotori.php
~~~

## Usage

```php
<?php
require './Kotori.php';

$app = new Kotori();

$config['APP_PATH'] = './app/';

$app->run();
```
