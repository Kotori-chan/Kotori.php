## 简介

KotoriFramework 是一个免费开源的，快速、简单的面向对象的 轻量级PHP开发框架 ，遵循Apache2开源协议发布。

## 全面的WEB开发特性支持

最新的KotoriFramework为WEB应用开发提供了强有力的支持，这些支持包括：

*  VC支持-基于视图（V）、控制器（C）的设计模式通过REST控制器扩展提供了RESTFul支持，为你打造全新的URL设计和访问体验
*  云平台支持-提供了对新浪SAE平台的强力支持，具备“横跨性”和“平滑性”，支持本地化开发和调试以及部署切换，让你轻松过渡，打造全新的开发体验。

## 大道至简的开发理念

无论从底层实现还是应用开发，我们都倡导用最少的代码完成相同的功能，简单和快速开发是我们不变的宗旨。

## 安全性

框架在系统层面提供了众多的安全特性，确保你的网站和产品安全无忧。这些特性包括：

*  XSS安全防护
*  输入数据过滤
*  防SQL注入

## 商业友好的开源协议

遵循Apache2开源协议发布。Apache Licence是著名的非盈利开源组织Apache采用的协议。该协议和BSD类似，鼓励代码共享和尊重原作者的著作权，同样允许代码修改，再作为开源或商业软件发布。
